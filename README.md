# Break the Silence 🌟

A powerful youth mental health movement in Kiambu County, Kenya, combining community support with innovative animal-assisted therapy through rabbit farming.

## Overview

Break the Silence addresses the silence surrounding mental health among youth in Kiambu County. We combine community support with innovative animal-assisted therapy through rabbit farming. Caring for rabbits calms the mind, restores routine, and rebuilds self-worth.

## Features

- Multi-page website structure with dedicated sections
- Responsive design for all devices
- Rabbit therapy information and resources
- Mental health resources page
- Contact form with WhatsApp integration
- Modern UI with Tailwind CSS

## Setup

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd BREAK-THE-SILENCE
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Create a `.env` file in the root directory and add your Firecrawl API key:
   ```
   FIRECRAWL_API_KEY=your_api_key_here
   ```

5. Start the development server:
   ```
   npm start
   ```

## API Integration

This project uses the Firecrawl API to fetch mental health resources. To use the API, you need to obtain an API key from [Firecrawl](https://www.firecrawl.dev/).
